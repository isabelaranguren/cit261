Guide to HTML5’s Media Tags – Audio and Video

The audio Tag

The audio tag allows you to put a music file directly into the HTML code without the use of a plugin,
such as Flash or Javascript. There are three types of audio formats that are compatible with this tag:
ogg, .mp3 and .wav.

Example Code

        <audio src="isabelaranguren.github.io/media/e856fe0f-5e1e-4f43-8be8-0539fbc2fa7b.mp3" controls="controls">
            Your browser does not support the audio element</audio>
