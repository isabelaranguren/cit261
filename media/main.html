Guide to HTML5’s Media Tags – Audio and Video

The <audio> Tag

The audio tag allows you to put a music file directly into the HTML code without the use of a plugin,
such as Flash or Javascript. There are three types of audio formats that are compatible with this tag:
ogg, .mp3 and .wav.

Example Code

        <audio src="audio.mp3" controls="controls">
            Your browser does not support the audio element</audio>
   

The <video> Tag

There are only three video formats that are supported with the <video> tag, they are:
.ogg, MPEG4 and WebM.

Example code

            <video id="demo" width="460">
            <source src="audio.mp3" type="video/mp4">
                Your browser does not support the audio element>
            </video>
